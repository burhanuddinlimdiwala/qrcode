📖 Quick Setup Guide
  1. Open Google Apps Script (script.google.com) → New project
  2. Paste the contents of Code.gs and save
  3. Change SHOP_SECRET to a strong secret key
  4. Run setupSheet() function once (authorize permissions)
  5. Click Deploy → New deployment → Web App
  6. Set "Execute as: Me" and "Who has access: Anyone" → Deploy
  7. Copy the Web App URL and paste it above alongside your secret key
  8. Set the Redeem Page URL to wherever you host redeem.html
